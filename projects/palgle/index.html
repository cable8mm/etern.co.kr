변환 후 Jekyll 도구를 이용해서 무료 Github Pages로 호스팅

복구 노트

워드프레스 플러그인을 거치지 않고 자체 제작한 Node.js 프로그램을 활용해 1,000여 개의 포스트와 메타데이터의 100% 무결성을 유지하며 추출했습니다.

추출된 DB 텍스트 데이터를 YAML Front Matter가 완벽히 규격화된 마크다운(Markdown) 포맷으로 깨끗하게 변환 및 포매팅했습니다.

Jekyll 정적 사이트 빌더를 도입하고 GitHub Actions를 결합하여 GitHub Pages에 정적 HTML 파일이 무료로 고속 자동 배포되는 완전 무비용 웹 인프라를 구축했습니다.

기존 포스트 내 깨진 절대 경로와 상대 링크를 전수 전처리하여 복원했으며, 유실되거나 깨진 이미지는 저장소 CDN에 호스팅하고 미존재 이미지는 텍스트로 보완 조치했습니다.
