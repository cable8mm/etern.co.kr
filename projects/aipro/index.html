복원 완료

스마트 물류 시스템

상품 12,000개, 연결 쇼핑몰 17개를 처리하는 대규모 스마트 물류 시스템을 복원했습니다.

과거 스택
PHP 7.3 / CakePHP 2 / MySQL 5.7
복원 스택
PHP 8.3 / Laravel 12 / MySQL 8

복구 노트

과거 사이트에서 사용하지 않는 필드와 테이블을 제거하고 대량의 상품 데이터를 효율적으로 정제하여 안정적으로 마이그레이션했습니다.

기존 CakePHP 2 + 스케폴딩 구조에서 Laravel Nova를 활용한 백오피스 어드민 시스템으로 새롭게 교체하여 유지보수성을 극대화하고 백엔드 UI를 현대화했습니다.

PHP 7.3에서 PHP 8.3, CakePHP 2에서 Laravel 12, MySQL 5.7에서 MySQL 8.0으로 전체 핵심 프레임워크와 런타임 버전을 업그레이드했습니다.

초기 데이터 시드를 구축하고 모든 핵심 화면의 자동화 테스트 코드를 작성하여 무결성을 안정적으로 보장했습니다.

Laravel Cloud의 지능형 자동 배포 엔진을 연동하여 향후 트래픽 증가 및 대용량 데이터 유입 시에도 스케일업이 가능한 유연한 클라우드 인프라를 마련했습니다.

오픈소스 데이터베이스 툴인 xeed(https://github.com/cable8mm/xeed)를 자체 개발 및 응용하여 테이블 구조별 모델, 시드, 팩토리, 마이그레이션 파일을 자동 설계해 엔지니어링 리소스를 절감했습니다.
