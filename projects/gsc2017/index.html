복구 노트

WordPress 사이트의 데이터를 크롤링하여 모든 페이지를 수집하고 깨진 링크와 이미지들을 완전하게 복구했습니다.

Visual Studio Code, Docker, Git을 결합한 테스트 환경을 설계해 원본 크롤링 데이터의 렌더링 무결성을 완벽히 검증했습니다. 이 테스트 컨테이너 내에서 소스코드를 미세 조정하고 변경 이력을 Git으로 철저히 버전 관리했습니다.

서버 유지보수 및 호스팅 비용을 0원으로 줄이기 위해 최종 복원 아티팩트를 정적 파일로 컴파일하여 GitHub Pages 환경에 안정적으로 배포했습니다. 데이터베이스 의존성이 없는 완전 무비용 인프라를 달성했습니다.

정적 페이지 형태의 시스템임에도 불구하고 GitHub Issue 및 Pull Request 프로세스를 도입하여 개정 이력을 체계적으로 보존하였으며, GitHub Actions CI/CD를 연동해 원클릭 자동 배포 파이프라인을 구축했습니다.
