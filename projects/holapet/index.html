올라펫 유기동물 플랫폼

올라펫의 버려진 유기동물 입양 서비스를 Github Actions를 이용한 클라우드 배포 환경으로 이전하여 다시 구동 가능하게 만들었습니다.

원본 기술 환경
PHP 7.3 / Laravel 8 / Bootstrap 3 / MySQL 5.7
장애 상태
Laravel에서 기본 CSS 프레임워크가 지원 종료됨
복원 작업
최신 버전으로 업그레이드 하고 Bootstrap 코드를 Tailwindcss로 변경
복원 결과
Laravel 최신 버전 이용이 가능해 졌고, 외부 라이브러리 이용 가능해 짐

복구 노트

과거 사이트에서 사용하지 않는 필드와 테이블을 제거하고 보호소 및 입양 공고글 데이터를 성공적으로 마이그레이션했습니다.

UI 프레임워크를 구형 Bootstrap 3에서 현대적인 Tailwind CSS 3로 완전히 변경하여 현대화했습니다.

PHP 7.3에서 PHP 8.3, Laravel 8에서 Laravel 12, MySQL 5.7에서 MySQL 8.0으로 코어 런타임을 업그레이드하고 전체 기능 테스트 코드를 작성해 무결성을 확보했습니다.

Github Actions를 활용한 지속적 통합/지속적 배포(CI/CD) 자동화 파이프라인을 구축하여 수동 배포 과정을 완전히 제거했습니다.
