# NOT_TO_DO.md

# Next State Studio — Anti-Scope & Engagement Policy

Version 1.0

---

# Purpose

Next State Studio는 모든 프로젝트를 수주하지는 않습니다.

우리는 제한된 리소스를 가장 가치 있는 문제 해결에 집중함으로써,
고객에게 더 높은 품질과 더 높은 성공 확률을 제공합니다.

이 문서는 우리가 수행하지 않는 업무와
프로젝트 수락 기준을 명확히 정의하기 위한 내부 운영 원칙입니다.

---

# Core Principle

우리는 새로운 서비스를 만드는 회사가 아닙니다.

우리는 이미 운영 중인 비즈니스가

- 더 안전하게
- 더 유지보수 가능하게
- 더 확장 가능하게

진화하도록 돕는 회사입니다.

모든 프로젝트는 이 원칙에 부합해야 합니다.

---

# 1. Anti-Scope

## 신규 기업 홈페이지 제작

### 수행하지 않습니다

회사 소개 사이트,
브랜드 웹사이트,
홍보 목적의 일반 홈페이지 제작

### 이유

이러한 프로젝트는 Next State Studio의 핵심 역량인

- 시스템 분석
- 데이터 보호
- 마이그레이션
- 현대화
- 안정화

와 직접적인 연관성이 없습니다.

### 예시

- 회사 홈페이지 제작
- 병원 홈페이지 제작
- 학원 홈페이지 제작
- 일반 기업 웹사이트 구축

---

## 마케팅 랜딩페이지 제작

### 수행하지 않습니다

광고 캠페인용 랜딩페이지,
이벤트 페이지,
프로모션 페이지 제작

### 이유

Next State Studio는 디자인 및 퍼블리싱 전문 스튜디오가 아닙니다.

우리는 운영 중인 서비스의
아키텍처, 데이터, 인프라 문제 해결에 집중합니다.

### 예시

- 이벤트 페이지 제작
- 광고 전환용 랜딩페이지 제작
- 단기 프로모션 페이지 제작

---

## 단순 퍼블리싱 업무

### 수행하지 않습니다

기존 시스템 개선과 무관한

- HTML 수정
- CSS 수정
- 배너 교체
- 문구 변경

등의 반복 작업

### 이유

해당 업무는 전문 퍼블리셔 또는 운영 담당자가
더 효율적으로 수행할 수 있습니다.

---

## 상주 개발 및 인력 파견

### 수행하지 않습니다

고객사에 상주하여

- 개발 인력 역할 수행
- 내부 팀 대체
- 장기 파견

형태로 일하는 계약

### 이유

Next State Studio는 외부 파트너로서
문제 해결과 구조 개선을 담당합니다.

특정 조직의 인력 공백을 메우는 역할은 수행하지 않습니다.

---

## Staff Augmentation

### 수행하지 않습니다

특정 기술 스택의 개발자를
기간 단위로 공급하는 형태의 계약

### 이유

우리는 인력을 판매하지 않습니다.

결과와 책임을 제공합니다.

---

# 2. Commercial Policy

## 최저가 경쟁 참여 금지

### 원칙

가격이 아닌 결과와 책임으로 경쟁합니다.

### 수행하지 않는 방식

- 최저가 입찰
- 단순 견적 비교 경쟁
- 무리한 가격 협상

### 이유

Migration,
Modernization,
Stabilization 프로젝트는

안전성과 경험이 핵심 가치이며
가격만으로 비교될 수 없습니다.

---

## Assessment 없는 본 프로젝트 착수 금지

### 원칙

진단 없이 실행하지 않습니다.

### 이유

현재 상태를 모르면

- 일정
- 비용
- 리스크

를 정확히 예측할 수 없습니다.

모든 프로젝트는 반드시

**Technical Assessment**

또는 이에 준하는 사전 진단 과정을 거쳐야 합니다.

---

## 책임 범위를 벗어난 확약 금지

### 원칙

통제할 수 없는 영역은 보장하지 않습니다.

### 예시

- 제3자 솔루션 장애
- 외부 API 사업자 이슈
- 앱스토어 정책 변경
- 고객 내부 운영 실수

# 4. Qualification Criteria

다음 조건을 충족하는 고객과 가장 좋은 결과를 만들 수 있습니다.

## 적합한 고객

- 운영 중인 서비스가 존재한다
- 데이터 보호가 중요하다
- 구조 개선 의지가 있다
- 장기적 관점으로 의사결정한다
- 품질과 안정성을 중요하게 생각한다

---

## 적합하지 않은 고객

- 가장 저렴한 견적만 찾는다
- 진단 없이 즉시 개발을 원한다
- 구조 개선보다 단기 처방만 원한다
- 반복적인 단순 운영 업무를 위임하려 한다

# Standard Response

대표님, 문의 주셔서 감사합니다.

다만 Next State Studio는 신규 홈페이지 제작이나 디자인·퍼블리싱 중심 프로젝트보다는 운영 중인 서비스의 마이그레이션, 현대화, 안정화 프로젝트에 집중하고 있습니다.

문의 주신 내용은 해당 분야 전문 업체가 더 적합할 것으로 판단됩니다.

좋은 파트너를 만나 프로젝트가 성공적으로 진행되길 바랍니다.
