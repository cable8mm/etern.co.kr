Next State Studio는

# Name Meaning

모든 서비스에는 현재 상태(Current State)가 있습니다.

그리고 모든 서비스는 언젠가 변화를 요구받습니다.

- 쇼핑몰 이전
- PHP 업그레이드
- WordPress 현대화
- 모바일 앱 개선
- 서버 이전
- 운영 환경 개선

하지만 변화는 위험합니다.

Next State Studio는

현재 상태를 이해하고

더 건강한 다음 상태(Next State)로 이동시키는 일을 합니다.

# Core Philosophy

## Understand Before Change

변경 전에 이해한다.

서비스를 충분히 이해하지 못하면

안전한 변화는 불가능하다.

---

## Data First

데이터는 가장 중요한 자산이다.

회원

주문

포인트

콘텐츠

운영 이력

모두 보호되어야 한다.

---

## Safe Transition

변화는 신중해야 한다.

업그레이드
