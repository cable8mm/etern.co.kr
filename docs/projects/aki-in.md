- 복구 노트 내용:
  1. 데이터베이스 설계
  - 과거 사이트에서는 사용하지 않는 필드와 테이블을 제거하고 마이그레이션하는 과정이 필요했습니다.
  - 무료 게시판 데이터베이스를 관계형 테이터베이스로 재설계하고 Admin UI 를 새롭게 구성했습니다.
  2. 프레임워크 변경
  - 과거 사이트에서는 Zeroboard 4.x 를 사용했으나, 현재는 CakePHP 2를 사용하여 어드민을 구성했습니다.
  - 이를 통해 유지보수성을 높이고, 관리자 화면의 UI를 현대화했습니다.
  3. PHP, MySQL 버전 업
  - PHP 5.6 to PHP 8.2
  - MySQL 5.7 to MySQL 8.0
  - 초기 데이터 시드를 만들고 모든 화면의 테스트 코드를 작성하여 서버 오픈과 배포할 때의 무결성을 보장합니다.
  4. Github 를 이용한 소스 코드 관리
  - Github 의 Private Repository를 이용하여 소스 코드를 관리합니다.
  - 이를 통해 소스 코드를 안전하게 보관하고, 변경 이력을 추적할 수 있습니다.
  5. 아이폰 및 안드로이드 API 서버 튜닝
  - 아이폰 및 안드로이드 API 서버를 튜닝했습니다.
  - 기존 레거시 API 코드의 속도를 개선하고, 데이터베이스 쿼리를 최적화했습니다.
  - 모바일 앱 사용자들이 더 빠르고 안정적으로 서비스를 이용할 수 있도록 성능을 개선했습니다.
