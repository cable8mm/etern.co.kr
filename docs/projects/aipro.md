- 프로젝트 영문: key aipro
- 프로젝트 제목: 스마트 물류 시스템 복원
- 과거 제작 연도: 2022
- 복원 완료 연도: 2026
- 과거 기술 스택: PHP 7.3 / CakePHP 2 / MySQL 5.7
- 복원 기술 스택: PHP 8.3 / Laravel 12 / MySQL 8
- 현재 상태: 복원 완료
- 이미지 파일명: assets/aipro-screenshot.png
- 한 줄 요약: 상품 12,000개, 연결 쇼핑몰 17개를 처리하는 물류 시스템을 복원했습니다.
- 복구 노트 내용:
  1. 데이터베이스 설계
  - 과거 사이트에서는 사용하지 않는 필드와 테이블을 제거하고 마이그레이션하는 과정이 필요했습니다.
  - 대량의 상품 데이터를 어떻게 효율적으로 마이그레이션하는 것이 가장 큰 과제였습니다.
  2. 프레임워크 변경
  - 과거 사이트에서는 CakePHP 2 + 스케폴딩을 사용했으나, 현재는 Laravel Nova를 사용하여 어드민을 구성했습니다.
  - 이를 통해 유지보수성을 높이고, 관리자 화면의 UI를 현대화했습니다.
  3. PHP, CakePHP, MySQL 버전 업
  - PHP 7.3 to PHP 8.3
  - CakePHP 2 to Laravel 12
  - MySQL 5.7 to MySQL 8.0
  - 초기 데이터 시드를 만들고 모든 화면의 테스트 코드를 작성하여 서버 오픈과 배포할 때의 무결성을 보장합니다.
  4. Laravel Cloud 를 이용한 클라우드 배포
  - Laravel Cloud 의 자동 배포 기능을 이용하여 서버 오픈과 배포하는 환경을 구축했습니다.
  - 이를 통해 트래픽과 용량이 늘어나도 안정적으로 운영할 수 있는 환경을 구축했습니다.
  5. Github 를 이용한 소스 코드 관리
  - Github 의 Private Repository를 이용하여 소스 코드를 관리합니다.
  - 이를 통해 소스 코드를 안전하게 보관하고, 변경 이력을 추적할 수 있습니다.
  6. 외부 코드 제작
  - https://github.com/cable8mm/xeed 를 만들어 데이터베이스 테이블마다 필요한 모델, 시드, 팩토리 그리고 마이그레이션 파일을 자동 생성합니다.
  - 이를 통해 개발 생산성을 높이고, 코드 품질을 개선할 수 있습니다.
