  1. 프레임워크 변경
     - 과거 사이트에서는 Gnuboard를 사용했으나, 현재는 Gnuboard와 함께 CakePHP 2를 사용하여 어드민을 구성하였습니다.

  3. static pages 의 값을 Admin으로 관리할 수 있도록 개선
     - 과거 사이트에서는 static pages 의 값을 파일로 관리했으나, 현재는 Admin으로 관리할 수 있도록 개선했습니다.
     - 이를 통해 유지보수성을 높이고, 관리자 화면의 UI를 현대화했습니다.
  4. 시간표를 별도로 관리할 수 있는 관리 도구 추가
     - 특히 매월 업데이트 되는 시간표를 파일로 관리했으나, 현재는 Admin으로 관리할 수 있도록 개선하여 업데이트를 용이하게 만들었습니다.
